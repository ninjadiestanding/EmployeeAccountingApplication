# Employee Accounting Application

# Description

This application allows you to keep track of employees.

# Requirements

To run this project, you will need:

* Visual Studio 2022
* .NET 5

# Installation

1. Clone this repository to your local machine
2. Open Visual Studio
3. Open the cloned project in Visual Studio
4. In the Visual Studio menu, select **Build > Build Solution** to build the project

# Usage

1. After the project has been successfully built, in the Visual Studio menu, select **Debug > Start Debugging** to run the program
2. If everything goes well, you will see the program window and the employees table
3. Now you can keep track of employees
